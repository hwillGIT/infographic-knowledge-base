# Infographic Knowledge Base

> **HBR Data & Visuals — Complete Design Language Reference**

A comprehensive knowledge base documenting the Harvard Business Review's infographic design system: chart types, color palettes, typography rules, layout anatomy, design principles, and AI prompt templates for replicating the HBR editorial visualization style.

## Quick Start

To generate a chart in the HBR style using any AI image model:

1. Choose your chart type from [01-chart-types.md](./01-chart-types.md)
2. Pick the matching prompt template from [06-prompt-templates.md](./06-prompt-templates.md)
3. Append the **Style DNA block** from [07-style-dna.md](./07-style-dna.md)
4. Verify your output against the [08-anatomy-checklist.md](./08-anatomy-checklist.md)
