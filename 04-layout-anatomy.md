# Layout & Structure Anatomy

> Every HBR chart follows a consistent layout container with specific proportions and structural rules.

---

## Standard Chart Container

```
+----------------------------------------------------------+
| [Bold serif title, 2 lines max]                          |
|                                                          |
| [1-3 sentence explainer, sans-serif, gray]               |
|                                                          |
| [Sub-chart question or axis label, bold, smaller]        |
|                                                          |
| +------------------------------------------------------+ |
| |                  THE CHART ITSELF                    | |
| +------------------------------------------------------+ |
|                                                          |
| Source: Author Names, Organization, Year    [HBR shield] |
+----------------------------------------------------------+
  [Category tag]  [Download Chart]  [From: Article Title]
```

---

## Dimension Rules

| Property | Value |
|----------|-------|
| **Compact chart width** | ~360px (column-width) |
| **Full-width chart** | ~610px |
| **Orientation** | Portrait (tall relative to width) |
| **Chart bounding box** | None — floats on white background |
| **HBR shield logo** | Bottom-right corner, always present |
| **Source line** | Bottom-left, always present |

---

## Layer-by-Layer Structure

### Layer 1: Category Tag (above chart)
- Small, colored text in sans-serif
- Indicates topic area (e.g., "Generative AI", "Talent Management")
- Color matches the accent color used in the chart

### Layer 2: Chart Title
- Bold serif typeface
- 2 lines maximum
- States conclusion, not just variable
- Dark charcoal `#222222`
- 18–24px

### Layer 3: Subtitle / Explainer
- 1–3 sentences of interpretive framing
- Sans-serif, regular weight
- Medium gray color
- Adds context beyond the title

### Layer 4: Sub-chart Label
- Bold sans-serif, smaller text
- Describes specifically what is being shown (the metric)
- Often phrased as a question
- Left-aligned

### Layer 5: The Chart Itself
- White background
- No border or frame around the chart
- 4–5 dotted horizontal gridlines max (no vertical)
- Direct data labels on elements
- Flat fills — no gradients, no shadows

### Layer 6: Source Line
- Bottom-left of chart
- Format: `Source: Author Name(s), Organization, Year`
- Small italic sans-serif, 9–10px
- Medium gray color

### Layer 7: HBR Shield Watermark
- Bottom-right corner
- Present on every chart
- Small, unobtrusive

### Layer 8: Action Bar (below chart)
- Download chart link
- Article title attribution
- Category tag (sometimes repeated)

---

## Grid & Spacing Rules

- **Horizontal gridlines**: Dotted, 4–5 max, no vertical grid
- **No axis border boxes or chart frames**
- **No bounding box** around chart — floats on white
- **Internal padding**: Consistent, generous top/bottom margins inside chart area
- **Row separators**: Thin light gray horizontal rules (in tables/strip plots)

---

## Small Multiples Layout

When comparing multiple categories:
- **3×3 or 3×4 grid** of identical mini-charts
- **Same axes and scale** across all panels
- **Label at top** of each panel (gray, centered, bold)
- **Bold end-value callout** at the final data point
- Green area fill + bold red trend line per panel

---

## Multi-Panel Infographic Layout

For complex infographics with multiple data types:
- **TOP SECTION**: Primary chart (bar or comparative)
- **MIDDLE SECTION**: Time-series or line projection
- **RIGHT SECTION**: Isotype/icon array or ratio callout
- **BOTTOM SECTION**: 3-column stat card panel
- Sections separated by thin light gray horizontal rules
- All sections share white background
- Title spans full width at top
