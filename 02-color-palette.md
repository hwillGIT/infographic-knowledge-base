# Color Palette Reference

> HBR uses a restrained, corporate-editorial palette — never garish, always purposeful.

---

## Primary Data Colors

| Color Name | Hex Code | Usage |
|------------|----------|-------|
| **HBR Red** | `#D0021B` | Bold trend lines, "bad" columns, urgency indicators |
| **HBR Purple / Violet** | `#7B5EA7` or `#9B59B6` | Matrix quadrants, stacked bars for negative sentiment |
| **Lavender / Soft Purple** | `#C4A8D9` | Secondary bars, lighter fills |
| **HBR Teal / Sage Green** | `#3AADA5` or `#41B3A3` | Positive/HIGH indicators, dot series |
| **Warm Green** | `#4CAF50` / `#27AE60` | Positive data (wealth, engaged employees, "good revenue") |
| **Amber / Gold** | `#F5A623` / `#E8A838` | MODERATE/LOW indicators in heatmaps |
| **HBR Orange** | `#E87722` | Accent dots, expert evaluator series |
| **Cool Blue (muted)** | `#7FB3D3` or `#89CFF0` | Secondary comparison, ghost/remainder bars |
| **Magenta / Pink** | `#E91E8C` | S-curve trajectory lines, special callouts |

## Background & Structure Colors

| Color Name | Hex Code | Usage |
|------------|----------|-------|
| **White** | `#FFFFFF` | Primary chart background, always |
| **Off-white / Light ice blue** | `#EEF4F7` | Collection card backgrounds, era highlights |
| **Light gray** | `#F5F5F5` or `#E8E8E8` | "THE SAME" segments, neutral fills |
| **Medium gray** | `#D0D0D0` | "Neither" segments in diverging bars |
| **Near-black** | `#1A1A1A` / `#222222` | Body text, decision box backgrounds |
| **Medium gray (text)** | `#666666` | Subtitle text, axis labels, source lines |

---

## Color Rules

1. **Maximum 2–3 accent colors per chart** — never a rainbow
2. **Ghost/remainder bars**: shown in pale wash of the primary color (~15–20% opacity), e.g., light blue behind red bars
3. **Negative values or contrasting categories**: red; positive in green or teal
4. **Matrix quadrants**: use progressively lighter tints of the same hue moving from featured to non-featured quadrants
5. **Flat fills only**: zero gradients, no drop shadows, no 3D effects
6. **Axis accent labels**: in matching accent color (e.g., teal for positive axis, purple for strategic axis)

## Color-to-Meaning Mapping

| Meaning | Color |
|---------|-------|
| Urgency / Negative / "Bad" | HBR Red `#D0021B` |
| Featured / Strategic / Primary | HBR Purple `#7B5EA7` |
| Positive / HIGH / Growth | Teal `#3AADA5` |
| Moderate / Warning | Amber `#F5A623` |
| Secondary comparison | Muted Blue `#89CFF0` |
| Positive comparison | Warm Green `#4CAF50` |
| Third series / Expert | Orange `#E87722` |
| Neutral / Background fill | Light Gray `#E8E8E8` |
| Text / Dark boxes | Near-black `#222222` |

## Quadrant Color Schemes

### Standard 2×2 Matrix (Deep-to-Light Purple)
- **Top-right (high-high)**: Deep purple `#6B46A0`
- **Top-left (high-low)**: Medium teal `#3AADA5`
- **Bottom-right (low-high)**: Medium lavender `#9B7DC8`
- **Bottom-left (low-low)**: Light sage `#7EC8C0`

### Scatter Positioning Map (20% opacity zones)
- **Top-right**: Teal `#3AADA5` @ 20% opacity
- **Top-left**: Amber `#F5A623` @ 20% opacity
- **Bottom-right**: Lavender `#C4A8D9` @ 20% opacity
- **Bottom-left**: Sage green `#8FBC8F` @ 20% opacity

## Bubble Dot Matrix Column Colors

For multi-column bubble matrices, use one distinct color per column:
- Green `#4CAF50`
- Amber `#F5A623`
- Blue `#5B9BD5`
- Purple `#9B59B6`
- Gray `#8E8E8E`
- Orange `#E87722`
